# 青龙面板文档 已测试v0.5.0

  • 默认配置文件名 cat_bili_config.json，或 .json5
  • 配置文件名也可以通过环境变量 BILITOOLS_FILE_NAME 指定，后缀 .json 可以省略
  • 由于作者并不熟悉，建议直接看青龙面板教程 https://github.com/whyour/qinglong在新窗口打开

# 使用 npm 包

安装 Node 依赖（侧边栏中依赖管理）

@catlair/bilitools

然后拉取青龙面板运行的代码

ql raw https://gitee.com/Kudouran/BiliTools/raw/main/tools/bilitools_npm.js

ql raw https://raw.githubusercontent.com/Kudouran/BiliTools/main/tools/bilitools_npm.js

require('@catlair/bilitools').ql();

手动创建一个 bilitools_npm.js 文件，并且把上面的代码拷贝进去

然后手动创建一个任务， task bilitools_npm.js

创建一个配置文件 cat_bili_config.json 配置与其他相同

或者使用环境变量

# 更新 npm 包

自动更新如下所示：

pnpm remove -g @catlair/bilitools && pnpm add -g @catlair/bilitools

PS: 可以不执行 pnpm remove -g @catlair/bilitools，不过前一个版本就会一直保留在磁盘中。

# 配置文件

满足以下的目录结构的 cat_bili_config.json{5} 都是符合要求的配置文件路径。

├── config
│  ├── cat_bili_config.json
│  └── cat_bili_config.json5
└── src
   ├── config
   │  ├── cat_bili_config.json
   │  └── cat_bili_config.json5
   ├── cat_bili_config.json
   ├── cat_bili_config.json5
