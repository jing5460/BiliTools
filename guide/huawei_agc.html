# 华为 AGC 云函数文档 已测试v0.4.0

下载最新版的 huawei_agc.zip 压缩包在新窗口打开

Gitee 备份下载地址在新窗口打开

加速下载：

https://ghproxy.com/https://github.com/Kudouran/BiliTools/releases/download/v0.5.7/huawei_agc.zip在新窗口打开

把 v0.5.7 替换成最新版本号即可

# 2. 新增 AGC CF

点击进入 AGC在新窗口打开

先创建一个项目，打开项目后开通云函数

开通云函数后创建新的云函数，注意选择地域，后面不能改

# 基础配置

  • 运行环境选择最新的 Nodejs（最低请选择 14）

创建好后修改基本信息

# 高级配置

环境变量名为 BILITOOLS_CONFIG

配置就是 gzip 压缩后的，可以在此处压缩，压缩详见在新窗口打开

# 触发器配置

此处使用 CRON 表达式是 UTC 时间，本地时间需要小时减 8

记得开启触发器

在 GitHub 上编辑此页 在新窗口打开
